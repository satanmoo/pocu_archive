## 메모이제이션을 사용한 피보나치 함수

```java
private static int fibonacci(final int number) {
    final int[] memo = new int[number + 1];
    Arrays.fill(memo, -1);

    return fibonacciRecursive(number, memo);
}

private static int fibonacciRecursive(final int number, final int[] memo) {
    if (number <= 1) {
        return number;
    }

    if (memo[number] > 0) {
        return memo[number];
    }

    int value = fibonacciRecursive(number - 1, memo) + fibonacciRecursive(number - 2, memo);
    memo[number] = value;

    return value;
}
```

## top-down DP

- 메모이제이션과 함께 사용하는 경우가 많음
- 하위 문제를 평가하는 순서가 중요하지 않음
    - 최적의 순서가 반드시 필요한건 아님
- 대표적인 예시가 피보나치 수열

## bottom-up DP의 장점

- CPU 캐시 친화적
    - 직전에 계산한 값 이용
    - 배열의 연속된 값 이용
- 재귀 호출을 피할 수 있음

## top-down DP가 bottom-up Dp 보다 유리한 경우

- 모든 하위 문제를 풀 필요 없는 경우
    - bottom-up은 모든 하위 문제를 해결하는 것을 강제하기 때문에 불리함

## memoization, tabulation 비용

- 메모리(공간)
- CPU가 더 비싼 자원이기 때문에 일반적으로 속도를 향상시키는 memoization과 tabulation을 선호

## 배낭문제 시간 복잡도

## 배낭문제 DP 코드

```java
import java.util.Arrays;

public class Program {

    public static void main(String[] args) {
        Item[] items = {
                new Item(5, 5),
                new Item(2, 4),
                new Item(6, 11),
                new Item(4, 6)
        };
        var res1 = getMaxValueBottomUp(15, items);
        System.out.println(res1);

        var res2 = getMaxValueTopDown(15, items);
        System.out.println(res2);
    }

    private static int getMaxValueTopDown(final int numSpace, final Item[] items) {
        final int[][] memo = new int[items.length][numSpace + 1];
        Arrays.stream(memo).forEach(it -> Arrays.fill(it, -1));

        return getMaxValueTopDownRecursive(numSpace, items.length - 1, memo, items);
    }

    private static int getMaxValueTopDownRecursive(final int currentSpace, final int itemIndex, final int[][] memo, final Item[] items) {
        if (itemIndex < 0 || currentSpace <= 0) {
            return 0;
        }

        if (memo[itemIndex][currentSpace] >= 0) {
            return memo[itemIndex][currentSpace];
        }

        int v1 = getMaxValueTopDownRecursive(currentSpace, itemIndex - 1, memo, items);
        int v2 = getMaxValueTopDownRecursive(currentSpace - items[itemIndex].space, itemIndex - 1, memo, items) + items[itemIndex].value;
        memo[itemIndex][currentSpace] = Math.max(v1, v2);

        return memo[itemIndex][currentSpace];
    }

    private static int getMaxValueBottomUp(final int numSpace, final Item[] items) {
        int[][] grid = new int[items.length][numSpace + 1];

        for (int i = 0; i < items.length; ++i) {
            final Item item = items[i];
            for (int j = 1; j <= numSpace; ++j) {
                if (i == 0) {
                    if (item.space > j) {
                        grid[i][j] = 0;
                    } else {
                        grid[i][j] = item.value;
                    }
                } else {
                    if (item.space > j) {
                        grid[i][j] = grid[i - 1][j];
                    } else {
                        grid[i][j] = Math.max(grid[i - 1][j], grid[i - 1][j - item.space] + item.value);
                    }
                }
            }
        }

        return grid[items.length - 1][numSpace];
    }

    private static class Item {
        int value;
        int space;

        public Item(final int value, final int space) {
            this.value = value;
            this.space = space;
        }
    }
}
```

## DP를 적용할 수 있는 문제의 특징

- 최적 부분 구조
    - 하위 문제의 최적 해법으로 상위 문제의 최적 해법을 구할 수 있음
- 하위 문제의 반복
    - 메모이제이션
    - 반복이 많을 수록 계산을 줄일 수 있어서 유리함

## 분할 정복과 DP 비교

- 하위 문제가 반복되는지 여부
    - 분할 정복은 반복되지 않음

## 그리디와 DP 비교

- 그리디에서 최적 부분 구조가 존재하면, local 선택이 global 최적 선택으로 이어짐
- DP는 모든 하위 문제를 고려하고, 중복 계산을 피해 global 최적 해를 구하려고 함

- 그리디는 한 번 선택하면 그 이후 되돌아가지 않음
    - 현재 선택이 미래에 끼칠 부정정인 영향을 고려하지 않음
- DP는 모든 하위 문제를 고려해 최적 해를 구함
    - 현재 선택이 미래애 끼치는 부정적인 영향을 고려해 계산

    public static void main(String args[]) {
        List<Schedule> scheduleList = new ArrayList<>();
        scheduleList.add(new Schedule("경찰서", LocalTime.of(8, 0), LocalTime.of(10, 0)));
        scheduleList.add(new Schedule("은행", LocalTime.of(9, 0), LocalTime.of(12, 0)));
        scheduleList.add(new Schedule("병원", LocalTime.of(9, 0), LocalTime.of(11, 0)));
        scheduleList.add(new Schedule("박물관", LocalTime.of(10, 0), LocalTime.of(13, 0)));
        scheduleList.add(new Schedule("시청", LocalTime.of(9, 0), LocalTime.of(14, 0)));
        scheduleList.add(new Schedule("골드문 본사", LocalTime.of(11, 0), LocalTime.of(15, 0)));
        scheduleList.add(new Schedule("주유소", LocalTime.of(12, 0), LocalTime.of(13, 0)));
        scheduleList.add(new Schedule("도서관", LocalTime.of(13, 0), LocalTime.of(16, 0)));
        scheduleList.add(new Schedule("편의점", LocalTime.of(14, 0), LocalTime.of(15, 0)));
        scheduleList.add(new Schedule("전당포", LocalTime.of(16, 0), LocalTime.of(19, 0)));
        scheduleList.add(new Schedule("POCU 대학교", LocalTime.of(15, 0), LocalTime.of(20, 0)));

        List<Schedule> result = findOptimalIntervalSchedule(scheduleList);
        for (var e : result) {
            System.out.println(e);
        }
    }

    private static List<Schedule> findOptimalIntervalSchedule(List<Schedule> scheduleList) {
        scheduleList.sort(Comparator.comparing(s -> s.end));
        List<Schedule> result = new ArrayList<>();

        Schedule current = scheduleList.get(0);
        result.add(current);
        for (int i = 1; i < scheduleList.size(); ++i) {
            Schedule next = scheduleList.get(i);
            if (next.start.isAfter(current.end)) {
                result.add(next);
                current = next;
            }
        }
        return result;
    }

    private static class Schedule {
        String name;
        LocalTime start;
        LocalTime end;

        public Schedule(String name, LocalTime start, LocalTime end) {
            this.name = name;
            this.start = start;
            this.end = end;
        }

        @Override
        public String toString() {
            return String.format("%s(%s~%s)", name, start, end);
        }
    }
}

## 허프만 인코딩/디코딩 코드

```java
import java.util.Comparator;
import java.util.HashMap;
import java.util.PriorityQueue;

public class Program {

    public static void main(String[] args) {
        {
            final String input = "banana*bab";
            final String encoded = encode(input);
            final Node hft = makeTree(input);
            String decoded = decode(encoded, hft);

            assert decoded.equals(input);
            assert encoded.length() == 19;
        }

        {
            final String input = "mommymamaoa";
            final String encoded = encode(input);
            final Node hft = makeTree(input);
            String decoded = decode(encoded, hft);

            assert decoded.equals(input);
            assert encoded.length() == 20;
        }
    }

    private static String decode(final String encoded, final Node root) {
        int start = 0;
        StringBuilder sb = new StringBuilder();
        while (start < encoded.length()) {
            var res = findRecursive(root, 0, start, encoded);
            sb.append(res.c);
            start += res.depth;
        }

        return sb.toString();
    }

    private static Result findRecursive(final Node node, final int depth, final int start, final String encoded) {
        if (Node.isLeaf(node)) {
            return new Result(node.ch, depth);
        }

        if (encoded.charAt(start + depth) == '0') {
            return findRecursive(node.left, depth + 1, start, encoded);
        }
        return findRecursive(node.right, depth + 1, start, encoded);
    }

    static class Result {
        char c;
        int depth;

        public Result(final char c, final int depth) {
            this.c = c;
            this.depth = depth;
        }
    }

    private static String encode(final String str) {
        final Node hft = makeTree(str);
        final HashMap<Character, String> memo = new HashMap<>();
        generateVariantCodeRecursive(hft, new StringBuilder(), memo);

        final var result = new StringBuilder();
        for (char c : str.toCharArray()) {
            result.append(memo.get(c));
        }

        return result.toString();
    }

    private static void generateVariantCodeRecursive(final Node current, final StringBuilder code, final HashMap<Character, String> codeMap) {
        if (Node.isLeaf(current)) {
            codeMap.put(current.ch, code.toString());
            return;
        }

        generateVariantCodeRecursive(current.left, code.append("0"), codeMap);
        code.deleteCharAt(code.length() - 1);
        generateVariantCodeRecursive(current.right, code.append("1"), codeMap);
        code.deleteCharAt(code.length() - 1);
    }

    private static Node makeTree(final String str) {
        final HashMap<Character, Integer> frequencies = new HashMap<>();
        for (char c : str.toCharArray()) {
            var old = frequencies.getOrDefault(c, 0);
            frequencies.put(c, old + 1);
        }

        final PriorityQueue<Node> pq = new PriorityQueue<>(Comparator.comparingInt(node -> node.frequency));
        for (var e : frequencies.entrySet()) {
            pq.add(new Node(e.getKey(), e.getValue()));
        }

        while (pq.size() > 1) {
            Node node1 = pq.poll();
            Node node2 = pq.poll();
            Node sum = Node.sum(node1, node2);
            pq.add(sum);
        }

        return pq.poll();
    }

    static class Node {
        char ch;
        int frequency;
        Node left;
        Node right;

        Node(final char ch, final int frequency) {
            this.ch = ch;
            this.frequency = frequency;
            left = null;
            right = null;
        }

        Node(final int frequency, final Node left, final Node right) {
            this.frequency = frequency;
            this.left = left;
            this.right = right;
        }

        static boolean isLeaf(final Node node) {
            return node.left == null && node.right == null;
        }

        static Node sum(final Node left, final Node right) {
            return new Node(left.frequency + right.frequency, left, right);
        }
    }
}
```

## 인터벌 파티셔닝

- 문제 해석:
    - 겹치는 인터벌끼리 그룹화
    - 그룹에 포함되는 인터벌의 수를 최소화하는 문제

- 풀이:
    - 시작 시간을 기준으로 인터벌을 오름차순 정렬
    - 앞에서부터 차례대로 인터벌을 뽑아 그룹에 포함할지 판단
    - 그룹은 우선 순위큐로 정렬 기준은 인터벌의 종료시간 오름차순
        - 인터벌 종료시간의 min-heap
    - 그룹에 최소 종료시간 인터벌(min-heap 가장 우선순위 높은 요소)의 종료시간(a)과 새로 뽑은 인터벌의 시작시간(b)을 비교
        - b < a 만족하면, 새로 뽑은 인터벌을 우선 순위 큐에 넣음
        - b >= a 만족하면, 우선순위 큐에서 가장 우선순위 높은 요소를 제거한 뒤 새로 뽑은 인터벌을 우선순위큐에 넣음
    - 모든 인터벌을 처리하고 우선순위 큐의 크기가 최소 그룹의 수

- 최적해:
    - 위 처럼 풀면 global 최적해를 만족

- 시간복잡도:
    - 정렬 비용 + 힙 비용
    - O(nlogn)

## Minimizing lateness

- 문제해석

- 일감(Task)는 처리시간과 마감시간을 속성으로 가진다.
- 한 번에 하나의 일만 처리할 수 있다고 가정

- 일감들을 처리할 때 "일감들의 지연시간"을 최소화하는 것이 목표
- 지연시간은 다음과 같이 정의함
    - 어떤 일의 지연시간을 "Max(0, 일 완료시간 - 일 마감시간)"로 계산하고
        - 지연시간이 음수가 될 수 없음
        - 빨리 끝내면 좋다는 개념
    - 각 일의 지연시간 중 최대값을 "일감들의 지연시간"으로 정의함

- 풀이:
    - 마감 시간의 오름차순으로 일감을 정렬하고 순서대로 뽑기

- 최적해:
    - 위 풀이는 최적해를 보장

- 예를 들어

| **작업** | **처리 시간** | **마감 시간** |
|--------|-----------|-----------|
| A      | 3         | 4         |
| B      | 2         | 5         |
| C      | 1         | 6         |

- A의 지연시간:
    - 0
    - 빨리 끝냈음
- B의 지연시간:
    - 0
    - 3에서 시작해서 처리하는데 2만큼 걸려서 5에서 끝남
    - Max(0, 5 - 5)
- C의 지연시간:
    - 0
    - 5에서 시작해서 처리하는데 1만큼 걸려서 6에서 끝남
    - Max(0, 6 - 6)

- 위 결과 "일감들의 지연시간"은 0으로 global 최적해를 구했음
- 시간복잡도는 정렬 비용

## 정수 목록이 있을 때 이 목록의 모든 요소들의 총합이 동일한 두 개의 집합으로 나눌 수 있는지 판단하는 문제 DP 점화식 구하기

- 총합을 S(짝수)라고 하면, i번째 요소까지 사용해 S/2를 만들면 나머지 요소로 S/2를 반드시 만들 수 있다
- "i번째 요소까지 사용해 S/2를 만들 수 는지 여부"로 문제를 다르게 해석할 수 있음
- i-1번째 요소까지 사용해 S/2를 만들 수 있으면, i번째 요소를 사용하지 않아도 "i번째 요소까지 사용해 S/2를 만들 수 는지 여부"는 참
- i-1번째 요소까지 사용해 S/2 - a(i번째 요소의 값)을 만들 수 있으면, i번째 요소를 추가해 "i번째 요소까지 사용해 S/2를 만들 수 는지 여부"는 참

- dp[i][S/2] = dp[i - 1][S/2] || dp[i - 1][S/2 - a]
    - dp 배열에는 boolean 값을 쓴다
