public class Program {

    public static void main(String[] args) {
        Node zero = new Node('0');
        Node one = new Node('1');
        Node two = new Node('2');
        Node three = new Node('3');
        Node four = new Node('4');
        Node five = new Node('5');

        zero.neighbors.add(two);
        two.neighbors.add(zero);

        one.neighbors.add(four);
        four.neighbors.add(one);
        one.neighbors.add(five);
        five.neighbors.add(one);

        two.neighbors.add(three);
        three.neighbors.add(two);
        two.neighbors.add(four);
        four.neighbors.add(two);

        four.neighbors.add(five);
        five.neighbors.add(four);

        searchBreathFirst(four);
    }

    static void searchBreathFirst(final Node node) {
        final HashSet<Node> discovered = new HashSet<>();
        final Queue<Node> queue = new LinkedList<>();

        discovered.add(node);
        queue.add(node);
        while (!queue.isEmpty()) {
            final Node current = queue.poll();
            System.out.println(current.data);
            for (final Node neighbor : current.neighbors) {
                if (!discovered.contains(neighbor)) {
                    discovered.add(neighbor);
                    queue.add(neighbor);
                }
            }
        }
    }

    private final static class Node {
        final char data;
        final List<Node> neighbors = new ArrayList<>();

        Node(final char data) {
            this.data = data;
        }
    }
}

import java.util.ArrayDeque;
import java.util.ArrayList;
import java.util.Deque;
import java.util.HashMap;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Program {

    public static void main(String[] args) {
        final Node a = new Node("집");
        final Node b = new Node("경찰서");
        final Node c = new Node("학교");
        final Node d = new Node("공원");
        final Node e = new Node("은행");
        final Node f = new Node("도서관");

        a.neighbors.add(b);
        b.neighbors.add(a);
        a.neighbors.add(d);
        d.neighbors.add(a);

        b.neighbors.add(c);
        c.neighbors.add(b);
        b.neighbors.add(e);
        e.neighbors.add(b);

        c.neighbors.add(f);
        f.neighbors.add(c);

        e.neighbors.add(d);
        d.neighbors.add(e);
        e.neighbors.add(f);
        f.neighbors.add(e);

        final Deque<Node> path = new ArrayDeque<>();

        final int distance = findShortestPath(a, f, path);

        final Node none = new Node("서로소");

        final int nonDist = findShortestPath(a, none, path);

        System.out.println(distance);
        System.out.println(nonDist);
    }

    static int findShortestPath(final Node from, final Node to, final Deque<Node> outPath) {
        final HashMap<Node, Node> prevs = new HashMap<>();
        final HashMap<Node, Integer> distances = new HashMap<>();
        final Queue<Node> queue = new LinkedList<>();

        prevs.put(from, null);
        distances.put(from, 0);
        queue.add(from);
        while (!queue.isEmpty()) {
            final Node current = queue.poll();
            final int currentDistance = distances.get(current);

            if (current == to) {
                Node prev = to;
                while (prev != null) {
                    outPath.addFirst(prev);
                    prev = prevs.get(prev);
                }
                return currentDistance;
            }

            for (final Node next : current.neighbors) {
                if (!distances.containsKey(next)) {
                    prevs.put(next, current);
                    distances.put(next, currentDistance + 1);
                    queue.add(next);
                }
            }
        }

        return -1;
    }

    private final static class Node {
        final String data;
        final List<Node> neighbors = new ArrayList<>();

        Node(final String data) {
            this.data = data;
        }
    }
}

## 다익스트라 인접 행렬 구현

- 무방향 그래프 가정
- 시간복잡도:
    - O(V^2)

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.Collections;
import java.util.List;

public class Program {

    public static void main(String[] args) {
        final UndirectedGraph graph = new UndirectedGraph(6);

        graph.addEdge(0, 1, 2);
        graph.addEdge(0, 2, 3);

        graph.addEdge(1, 3, 1);
        graph.addEdge(1, 5, 6);

        graph.addEdge(2, 3, 2);

        graph.addEdge(3, 4, 2);

        graph.addEdge(4, 5, 1);

        final List<Integer> path = new ArrayList<>();
        int distance = graph.findShortestPath(0, 5, path);
    }

    private final static class UndirectedGraph {
        final int numVertices;
        final int[][] adjacentMatrix;

        UndirectedGraph(final int numVertices) {
            this.numVertices = numVertices;
            this.adjacentMatrix = new int[numVertices][numVertices];
        }

        void addEdge(final int i, final int j, final int weight) {
            adjacentMatrix[i][j] = weight;
            adjacentMatrix[j][i] = weight;
        }

        private int findShortestPath(final int from, final int to, final List<Integer> outPath) {
            if (from == to) {
                outPath.add(from);
                return 0;
            }

            final int[] distances = new int[this.numVertices];
            Arrays.fill(distances, Integer.MAX_VALUE);
            final boolean[] visited = new boolean[this.numVertices];
            final int[] prev = new int[numVertices];

            prev[from] = -1;
            distances[from] = 0;
            for (int iter = 0; iter < numVertices; ++iter) {
                int minVertex = -1;
                int minDistance = Integer.MAX_VALUE;
                for (int i = 0; i < distances.length; ++i) {
                    if (visited[i]) {
                        continue;
                    }
                    final int distance = distances[i];
                    if (distance < minDistance) {
                        minDistance = distance;
                        minVertex = i;
                    }
                }

                if (minVertex == -1) {
                    break;
                }

                for (int j = 0; j < adjacentMatrix[minVertex].length; ++j) {
                    final int weight = adjacentMatrix[minVertex][j];
                    if (weight == 0 || visited[j]) {
                        continue;
                    }
                    if (distances[j] > distances[minVertex] + weight) {
                        distances[j] = distances[minVertex] + weight;
                        prev[j] = minVertex;
                    }
                }
                visited[minVertex] = true;
            }
            // 경로를 찾지 못함
            if (distances[to] == Integer.MAX_VALUE) {
                return -1;
            }

            int updateBehavior = to;
            while (updateBehavior >= 0) {
                outPath.add(updateBehavior);
                updateBehavior = prev[updateBehavior];
            }
            Collections.reverse(outPath);

            return distances[to];
        }
    }
}

```

import java.util.HashMap;

import java.util.List;

public class Program {

    static int nodeKeyNum = 0;

    public static void main(String[] args) {
        {
            final Node home = new Node("집");
            final Node policeOfficer = new Node("경찰서");

            final int minDist = getMinimumDistance(home.key, school.key, graph, path);

            B.neighbors.put(A, 4);
            B.neighbors.put(C, 11);
            B.neighbors.put(D, 8);

            D.neighbors.put(G, 7);
            D.neighbors.put(H, 4);

            E.neighbors.put(D, 2);
            E.neighbors.put(C, 7);
            E.neighbors.put(F, 6);

            F.neighbors.put(H, 2);
            F.neighbors.put(E, 6);

            final List<Node> graph = List.of(A, B, C, D, E, F, G, H, I);
            final LinkedList<Node> path = new LinkedList<>();
            final int minDist = getMinimumDistance(A.key, I.key, graph, path);
        }

    private static int getMinimumDistance(final int fromKey, final int toKey, final List<Node> graph, final LinkedList<Node> outPath) {
        final Node[] prev = new Node[graph.size()];
        final PriorityQueue<Candidate> pq = new PriorityQueue<>(Comparator.comparing(candidate -> candidate.distance));
        final int[] minDistances = new int[graph.size()];
        // 최적해 배열 초기화
        Arrays.fill(minDistances, Integer.MAX_VALUE);
        // 시작점 처리
        prev[fromKey] = null;
        minDistances[fromKey] = 0;
        pq.add(new Candidate(graph.get(fromKey), 0));
        // 다익스트라 알고리듬
        while (!pq.isEmpty()) {
            final Candidate candidate = pq.poll();
            // 최소값이 확정된 노드 스킵(방문한 노드 스킵)
            // 큐에 최소값이 확정되지 않은 stale 상태의 노드가 들어가는 단점이 있음
            if (candidate.distance > minDistances[candidate.node.key]) {
                continue;
            }

            if (candidate.node == graph.get(toKey)) {
                break;
            }

            assert minDistances[candidate.node.key] == candidate.distance;

            for (final var e : candidate.node.neighbors.entrySet()) {
                final Node to = e.getKey();
                final int distance = e.getValue();
                final Node from = candidate.node;
                final int newDist = minDistances[from.key] + distance;
                if (minDistances[to.key] > newDist) {
                    prev[to.key] = candidate.node;
                    minDistances[to.key] = newDist;
                    pq.add(new Candidate(to, newDist));
                }
            }
        }

        // 최단 경로 없음
        if (minDistances[toKey] == Integer.MAX_VALUE) {
            return -1;
        }

        Node current = graph.get(toKey);
        while (current != null) {
            outPath.addFirst(current);
            current = prev[current.key];
        }

        return minDistances[toKey];
    }

    private static class Candidate {
        final Node node;
        final int distance;

        Candidate(final Node node, final int distance) {
            this.node = node;
            this.distance = distance;
        }
    }

    private static class Node {
        final int key = nodeKeyNum++;
        final String name;
        final HashMap<Node, Integer> neighbors = new HashMap<>();

        Node(final String name) {
            this.name = name;
        }

        @Override
        public String toString() {
            return "Node{" +
                    "name='" + name + '\'' +
                    '}';
        }
    }
}

## 음수인 변이 있을 때 다익스트라 알고리듬

- 최적의 해를 찾을 수 없음
    - 로직이 방문 처리 == 최적의 해를 구했다고 확정 짓는 것
        - 따라서 한 번 방문하고는 더 이상 방문하지 않는 것이 가정
    - 다음 거리는 언제나 기존의 방문한 거리(부분 최적해) 이상이라고 가정하고 푸는 문제
        - 이게 성립하려면 양의 가중치만 가능

## A* 알고리듬 장점과 단점

- 장점
    - 불필요한 평가를 피해서 효율적
- 단점
    - 휴리스틱으로 해의 정확성을 보장하지 않음

## A* 알고리듬이 전역적인 최적의 해를 보장하지 못하는 이유

- 노드 n에서 부터 목적지까지 추정 거리를 기반으로 계산하기 때문
- "시작에서 부터 노드 n까지 거리(실제 값) + n에서 부터 목적지까지 추정 거리(추정치)"를 최소화하는 식으로 노드를 선택
    - f(n) 함수로 부름
- f(n)을 정확하게 평가하지 못하기 때문에 다음으로 방문하는 노드(확장)이 잘못되어 경로가 잘못될 수 있음
    - 전역해를 놓칠 수 있음

## A* 알고리듬과 다익스트라의 차이점(동작 원리와 함께)

1. 그래프에 있는 모든 노드의 g(n) 과 f(n)을 최대값으로 초기화
2. 시작점 처리
    - g(s) = 0, f(s) = g(s) + h(s) == h(s)
3. 시작 노드 s를 OPEN에 추가
    - OPEN은 우선 순위큐로 비교자는 f(n)
4. OPEN에서 최소 노드를 찾고 제거
5. 제한 노드의 각 이웃 m에 대해서 g(n) + dist(from n to m) < g(m) 만족하면 업데이트, OPEN에 m 추가
    - 다익스트라와 동일하게 경로에 대해 업데이트 g(n)으로 계산
    - 여기서 업데이트할 때 이미 방문한 노드도 방문함!
6. 목적지에 도달하거나 OPEN이 빌 때 까지 4-5 반복

## h(n) 평가 함수에 대한 이해

- h(n) == 0
    - 다익스트라와 동일함
- h(n) <= h'(n)
    - 추정 거리가 실제 거리보다 짧거나 같은 경우
    - h(n)이 admissible한 경우라고 표현
    - 모든 n에 대해서 h(n)이 admissible하면 A*가 최적 보장
    - 이 조건을 유지하도록 작성하는게 중요함
- h(n) << h'(n)
    - 추정 거리가 실제 거리보다 훨씬 작은 경우
    - A*가 더 많은 경로를 탐색하고 넓게 탐색함
    - 속도가 느려짐
    - 직관적으로 생각해보면 f(n) == g(n) + h(n) 식에서 h(n)의 영향이 극단적으로 줄어드는 상황, 다익스트라와 가까워짐
        - 속도가 느려지고, A*의 원래의 목적인 빠른 탐색은 약해짐
- h(n) == h'(n)
    - 이상적인 경우
    - 모든 노드에 대해서 성립하면 빠르고, 최적의 경로를 찾을 수 있음
        - 다익스트라보다 빠른데 최적

### 모든 h(n)이 admissible하면 A*는 최적을 보장함을 귀류법으로 증명

- 나중에 포스팅

## A*가 중복 방문을 허용하는 이유

- OPEN(우선 순위 큐)에서 비교자는 f(n)으로 추정치가 포함됨
- OPEN에서 뽑은 최소 f(n)을 가지는 노드를 처리하고 다른 노드를 방문했을 때 정확한 g(n)이라는 정보가 추가되어 이전의 계산한 f(n)이 정확하지 않음을 확인할 수 있음
    - 이전의 계산한 최소값이 정확하지 않음
- 따라서 중복으로 방문을 해서 f(n)값을 갱신해줘야함

### 특정 조건을 만족하면 노드를 한 번씩만 방문

## APSP(All-Pairs-Shortest-Path)는 왜 필요할까?

- SSSP(Single-Source-Shortest-Path)를 모든 소스(노드)에 대해서 실행해도 모든 짝의 최단 경로를 구할 수 있음
    - 하지만 느리다
- 모든 노드 쌍에 대한 최단 경로를 더 빠르게 찾도록 설계된 알고리듬이 APSP

## 플로이드 워셜 알고리듬의 핵심

- u -> v가 최단 경로
- u -> m -> v 중간의 거쳐가는 m노드에 대해서
    - u -> m가 최단경로
    - m -> v가 최단경로

- 경유지를 거치는 경로도 최단 경로라는 성질을 이용

## 플로이드 워셜 공식

- 그래프에 노드를 1 ~ N 번호를 매김
    - 0번은 결번(없는 노드)

- sp(i, j, k)를 정의:
    - from i to j까지 최단 경로
    - 1~k번 노드를 중간에 경유하는 것을 고려해서 구한 최단 경로
    - 1~k번 노드를 방문하는 것은 선택임

- DP 관계식 정의
    - sp(i, j, k) == minOf(sp(i, j, k-1), sp(i, k, k-1) + sp(k, j, k-1))

- 풀어서 설명하면:
    - 1~k번까지 경유지를 고려해 구한 from i to j 최단경로는
    - 1~k-1번까지 경유지를 고려해 구한 from i to j 최단경로와 새롭게 k번까지 경유지를 고려해 구한 from i to j 최단경로 중 최소값

## 플로이드 워셜 풀이

- 노드에 번호 매기기

- k == 0 일 때 grid 초기화
    - 인접행력과 동일
- k == 1부터 공식 적용해서 grid 업데이트

- 시간복잡도:
    - grid 1회 업데이트할 때 O(V^2)
    - grid를 V회 업데이트함
        - 1~k 에서 k <= V, k의 최대값은 v

- 공간복잡도:
    - grid가 끝
    - 1번 선언하고 계속 업데이트
    - O(V^2)
