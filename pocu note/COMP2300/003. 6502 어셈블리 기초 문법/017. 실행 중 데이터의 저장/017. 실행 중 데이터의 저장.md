## 실행 중 데이터의 저장

![[Pasted image 20260118011952.png]]

*load*, *store* 용어 많이 쓰이니까 기억하자.

![[Pasted image 20260118012049.png]]

"특정 주소로"의 의미: 
- CPU가 주소 핀에 어떤 값을 쏜다. 
- 주소 핀은 RAM과 연결되어 있고, RAM은 신호를 받는다.
	- 이 때 읽고/쓰고는 다른 핀에서 결정된다.
		- [[012. 핀 값 읽는 프로그램 작성하기#RWB]] 참고

![[Pasted image 20260118012708.png]]

"데이터를 읽어옴/덮어씀"의 의미:
- 데이터는 데이터 핀을 통해서 CPU와 RAM사이를 이동한다.

## 간단한 덧셈 예 - a 모드

![[Pasted image 20260118013045.png]]

```asm
; 3+12
clc             ; 받아올림 초기화
```
*clc*: Carry 플래그를 0으로

```asm
lda #3
```
A레지스터에 3 로드
- Immediate 모드

```asm
adc #$0C
```

A레지스터의 값(`0x03`) + 피연산자(`0x0C`, immediate 모드)  + Carry(`0x00`) 를 A 레지스터에 저장

```asm
sta $1000
```

A레지스터의 값(`0x0F`)를 주소 `0x1000`에 저장

```asm
; 2+1
lda #2
```

A레지스터에 2 로드

```asm
adc #1
```

A레지스터의 값(`0x02`) + 피연산자(`0x01`, immediate 모드) + Carry(`0x00`) 를 A 레지스터에 저장

```asm
sta $1001
```

A레지스터의 값(`0x03`)를 주소 `0x1001`에 저장

```asm
; (3+12)-(2+1)
sec             ; 받아내림 초기화
```

*sec*: Set Carry 1
- 빌림 없음

```asm
lda $1000
```

A레지스터에 `0x1000`에 저장된 값(`0x0F`)을 로드한다.

```asm
sbc $1001
```

A레지스터에 저장된 값(`0x0F`) - 주소 `0x1001`에 저장된 값(`0x0x`) - (1 - Carray(`0x01`)) 를 A레지스터에 저장

```asm
sta $1002
```

A레지스터에 저장된 값`0x0C`을 주소 `0x1002`에 저장
