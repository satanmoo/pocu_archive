6502 스펙에서 강제하는 메모리 페이지는 2장
- [[014. zp 주소지정 모드]]
- 스택 메모리
	- 주소의 범위는 `0x00~0xFF`
	- 이 때 페이지 넘버는 `0x01`이다.

이전에 본 [[014. ROM 주소 매핑]]도 페이지를 설계자 마음대로 사용한 것이다.
- 리셋 벡터를 포함한 주소
- 128개 페이지

화면에 보여줄 픽셀 배열, 디스크 I/O 버퍼 등도 메모리 페이지를 사용하기 때문에, 6502 CPU의 모든 주소를 ROM과 RAM에만 할당할 수는 없다.
- 요즘 CPU는 주소를 모두 ROM, RAM에 사용하고, 나머진 하드웨어에서 자체 제공하는 편이다.


[[014. ROM 주소 매핑]]에서 본 것처럼 RAM에 0으로 시작하는 메모리를 할당했었다. 
- 스택 메모리도 RAM에 포함된다는 점도 위의 근거가 된다.
