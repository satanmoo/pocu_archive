## 색인 레지스터 - X, Y

![[Pasted image 20260119012808.png]]

1. 배열의 색인 처럼 : 어떤 시작 주소(배열 0번째 포인터)와 X,Y 값을 인덱스처럼 사용한다.
2. `cpx`, `cpy`: compare 라는 뜻이다. x,y 레지스터의 값을 비교할 수 있다. for문에서 i값을 비교할 때 사용할 수 있다.
3. `tax`: transfer from *a레지스터* to *x레지스터*, 전송하는 opcode
4. `tsx` : transfer from stack to x, 스택 포인터를 x레지스터에 임시 저장할 수 있다.

![[Pasted image 20260119013238.png]]

x,y 레지스터가 메인 레지스터가 아닌 이유 : ALU의 연산에 사용이 불가능
