## 예: 문자열 복사하기

![[Pasted image 20260221004335.png]]

[[009. 소문자로 바꾸기]]에서 이어서 ROM의 문자열을 RAM에 복사하는 코드를 작성해보자

### 직접 구현한 문자열 복사하기

```asm
;---문자열을 복사하는 코드  
    .PROCESSOR 6502  
    .ORG $8000  
  
;---출력 변수  
dst=$1000  
  
;---스택 초기화  
    ldx #$FF  
    txs  
  
;---문자열 순회 전 변수 초기화  
    ldx #$FF          ; 문자열 인덱스 초기화  
  
.loop:  
    inx; 문자열 인덱스 증가  
    lda src,x       ; 문자열의 현재 문자 로드  
    sta dst,x       ; 현재 문자를 dst로 복사  
    bne .loop       ; 반복  
   .exit:  
    jmp .exit  
  
;---문자열  
    .ORG $C000,0  
src:  
    .BYTE "Hello, World!"  
    .BYTE 0 ; 문자열의 끝을 나타내는 널 문자
```

### 수업에서 제공하는 코드

![[Pasted image 20260221011152.png]]

```asm
lda str,x
sta dst,x
```

> [!tip] X 레지스터 재활용
> *X 레지스터* 의 값을 재활용하는 것이 포인트.
> 
> 끝까지 순회하다보면 *X 레지스터* 는 문자열의 길이와 같아진다. 
> 
> **문자열의 길이도 구할 수 있다!!**
