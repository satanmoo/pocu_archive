## 예: 소문자로 바꾸기

![[Pasted image 20260220185544.png]]

> [!NOTE]
> 입력값은 아스키코드 알파벳 라고 가정한다.

### 직접 작성한 코드

```asm
; 문자열을 소문자로 바꾸는 코드  
    .PROCESSOR 6502  
    .ORG $8000  
  
;--- 출력 변수  
output=$1000  
  
;--- 스택 초기화  
    ldx #$FF  
    txs  
  
;--- 문자열 순회 전 변수 초기화  
    ldx #0          ; 문자열 인덱스 초기화  
.loop:  
    lda str,x       ; 문자열의 현재 문자 로드  
    beq .exit       ; 널 문자(0)이 나오면 종료  
    ora #$20        ; 대문자를 소문자로 변환 (ASCII에서 대문자와 소문자의 차이는 0x20)    
    sta output,x       ; 변환된 문자 저장  
    inx             ; 다음 문자로 이동  
    jmp .loop       ; 반복  
.exit:  
    jmp .exit  
  
    .ORG $C000,0  
str:  
    .BYTE "Hello, World!"  
    .BYTE 0 ; 문자열의 끝을 나타내는 널 문자  
  
    .ORG $FFFC,0  
    .WORD $8000  
    .WORD $0000
```

### Read Only가 반영된 C코드 작성하기

![[Pasted image 20260220203817.png]]

> [!INFO] `const char* str`
> 컴파일러의 고마움
> - 컴파일러에서 오류를 준다.
> - 실수를 잡아준다.

> [!NOTE] C에서의 문자열 처리
> C에서 `const char* str` 은 Data section의 Read only 영역에 들어간다.

 ### 해결 방법
 
 따라서 ROM의 Read Only 문자열을 RAM에 복사한 뒤 바꿔야 한다.
