## signed 뺄셈의 가능한 조합

![[Pasted image 20260202021651.png]]

"양수 - 양수", "음수 - 음수" >> 절대 오버플로 안나죠?

[[010. 8비트 뺄셈#sbc]] 에서 "sbc"의 동작을 확인해볼까??

> V 플래그: signed 오버플로가 발생하면 켜짐
> - 결과가 [-128, 127] 범위를 넘어서면 켜짐

양수 중에 가장 작은 수는 `0x00000000`, 이 값이 "A 레지스터"의 값이라고 간주하자.
양수 중에 가장 큰 수는 `0x01111111`, 이 값이 M(피연산자)의 값이라고 간주하자.

직관적으로 그냥 빼도 -127이라 [-128, 127]의 범위를 벗어나지 않았다.

"오버플로우 플래그"의 정의를 적용해보자.

"sbc"를 수행하면 다음과 같다.
1. "캐리 플래그"는 1로 시작
2. "M의 부호를 뒤집음"
	- `0x1000_0000`
3. 뒤집힌 M에 "캐리 플래그" 값을 더함
	- `0x1000_0001`
4. A와 더함
	- `0x1000_0001`
	- "캐리 플래그"는 0이된다.
		- 받아 내림이 있었다는 의미

> 결과의 부호가 입력의 부호와 다르고 A 레지스터의 부호와 다를 때

"A 레지스터"의 부호는 양수, M의 부호는 양수로 보이나 사실은 여기서 입력은 "-M"(M의 2의 보수, 뒤집힌 M에 캐리 플래그를 더한 비트 패턴) 이라서 음수다! 따라서 두 입력의 부호가 달라서 오버플로우가 발생하지 않는다!

## N XOR V를 하는 방법

![[Pasted image 20260202022720.png]]

"V 플래그"를 피연산자로 가져와서 XOR 연산에 사용할 방법이 없다.

> sbc를 사용해 A - M을 계산

결과가 "A 레지스터"에 저장되기 때문에 A의 "bit 7"이 바로 N 플래그 값이다.

> V 플래그가 켜져 있다면

[[004. 고수준 조건문 옮기기]] 에서 본 "bvc", "bvs"를 사용하자

"V == 1" 이면 "bvs"(Branch Overflow Set) 지만, 반대로 해야하니 "bvc"(Branch Overflow Clear), "V == 1"을 만족하면 바로 밑의 "eor #$80" 실행

> eor \#$80

"A 레지스터"의 "bit 7"(N 플래그 값)과 1(V == 1 이라서 V의 역할을 한다)을 xor 하는 거죠? 결과 비트는 "A 레지스터"에 저장된다.
- [[020. 논리 연산]] 참고
- "A 레지스터"의 "bit 7"이 "N XOR V"의 값
- eor 결과 "bit 7"의 값은 "N 플래그"를 결정하죠?
- 따라서 "N XOR V"의 값이 "N 플래그"에 저장됬다!

> bpl $901B

"V == 0" 이면 "N XOR V"를 하기 위한 위의 연산("eor")이 필요없다. V == 0 일 때 XOR 연산을 하면 항상 자기 자신(N)이 나온다.
- 따라서 이미 "N XOR V"의 값이 "N 플래그"에 저장됬다!

그리고 "N 플래그"(N XOR V의 결과)에 따라 분기하면 된다.

"N 플래그"가 1일 경우, 즉 음수인 경우("bit 7"이 1) "A < M" 일 때 실행할 코드를 실행하면 되고, 반대인 경우 "A >= M"일 때 실행할 코드를 실행하도록 조건문을 구현한다.

"bpl"(branch on plus)을 사용하면 "N 플래그"값이 0 일 때 분기한다. 따라서 "bpl" 바로 아래에 "A < M" 일 때 실행할 코드를 넣으면 된다.

### 다음 코드를 읽고 질문에 답하세요.

```asm
lda #$?? 
cmp #$10
bxx label
```

#### 1. A가 $10과 같은 경우 분기하려면 bxx 대신 어떤 니모닉을 사용해야 하나요?

같으면 "Z 플래그"가 켜지겠죠? branch equal

정답: "beq"

#### 2. A가 $10 이상인 경우 분기하려면 bxx 대신 어떤 니모닉을 사용해야 하나요?

이상인 경우 "cmp" 연산에서 뺏을 때 받아내림이 없다. "C 플래그"가 켜진다.

branch c set

정답: bcs

#### 3. A가 $10 미만인 경우 분기하려면 bxx 대신 어떤 니모닉을 사용해야 하나요?

이상의 반대니까

branch c clear

정답: bcc
