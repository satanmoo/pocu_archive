### 플래그의 변화를 표로 정리하면

![[Pasted image 20260201232902.png]]

> A == M

"N 플래그": "bit 7"이 0이라서 켜지지 않음
"Z 플래그": 모든 비트가 0이라서 켜진다.
"C 플래그": 빼기 할 때 받아 내림이 발생하지 않았기 때문에 켜진다!

> A < M

"N 플래그": "bit 7"에 따라 다르다. 연산 결과에 따라 달라진다.
"Z 플래그": 모든 비트가 0이 아니라 켜지지 않음
"C 플래그": 빼기 할 때 받아 내림이 발생하여 때문에 켜지지 않는다.

> A > M

"N 플래그": "bit 7"에 따라 다르다. 연산 결과에 따라 달라진다.
"Z 플래그": 모든 비트가 0이 아니라 켜지지 않음
"C 플래그": 빼기 할 때 받아 내림이 발생하지 않았기 때문에 켜진다!

> [!NOTE] N, Z, C 플래그의 조합에 따라 비교연산 중 어떤 것이 참인지 판단할 수 있다!


![[Pasted image 20260201233902.png]]

> Z == 1

표에서 "A == M"이 확정이다.

> Z == 0

일단 "A == M"이 아님은 확실하다. 그리고 "A < M", "A > M" 둘 중 하나라 "A != M"이 확정이다.

> C == 0

 표에서 이 조건을 만족하는 결과는 "A < M" 뿐이라 확정이다.

> Z == 0 && C == 1

일단 "Z == 0"을 만족한 상태에서 "C == 1"을 만족하면 "A > M"이 확정이다.

![[Pasted image 20260201234406.png]]

> C == 1

"A >= M" 으로 합칠 수 있다
