참고로 `i` 는 뒤에서 배운다. 
 - 주소 지정 모드
 - implied (묵시적)
 - 뒤에 바이트를 읽어서 사용하지 않음

캐리 플래그를 끄면?
- 0으로 초기화 한다.
	- 끄는 동작은 0, 켜는 동작은 1

![[Pasted image 20251220014005.png]]

`#` 은 주소 지정 모드
- 즉시 모드
- 그 옆의 바이트의 값을 바로 레지스터에 제공
- 상수를 제공하는 의미

![[Pasted image 20251220014119.png]]

다음 바이트의 값은 `0x0A`
따라서 A 레지스터에 이 값을 load 

![[Pasted image 20251220014203.png]]

![[Pasted image 20251220014232.png]]

`#` 에 따라 다음 바이트 값을 읽어, 받아올림을 이용한 덧셈에 사용한다.

레지스터 A에 이미 `0x0A` 값이 올라와 있고, 여기에 `0x06`값을 더해야 한다.

![[Pasted image 20251220014355.png]]

메모리에 A 레지스터의 값을 저장한다.

`a` 는 주소 지정 모드
- 2바이트를 읽어서 주소를 구한다.

![[Pasted image 20251220014449.png]]

주소는 `0x2200`
- 리틀 엔디언
 - 오른쪽이 메모리가 주소가 커지는 방향
