## 1. RAM을 읽다가 RAM을 읽기

![[Pasted image 20260102005631.png]]

위에서 말한대로 "조건 13"은 이미 만족하니까 중요하지 않다.

"조건 12"를 해석해보면, 전 사이클에도 "CEB", "OEB" 모두 VL, 이번 사이클도 "CEB", "OEB" 모두 VL이다.
- "continously"

"Case1"은 "조건 12", "조건 13"을 만족한다. 따라서 "RAM을 읽다가 RAM을 읽기"는 "Case1"에 대응된다.

## 2. RAM에 쓰다가 RAM을 읽기

![[Pasted image 20260102010526.png]]

 "조건 12"를 만족하는가? 만족하네요.

"조건 13"은 이번 사이클에 읽기라서 H라서 만족하고

"RAM에 쓰다가 RAM 읽기"는 "Case1"에 대응된다.

## 3. RAM을 사용 안 하다 RAM 읽기

![[Pasted image 20260102010723.png]]

"조건 12"를 만족하지 못한다. 저번 사이클에 "CEB"가 H기 때문이다.

"조건 14"를 이해하려면 추가적인 설명이 필요하다.

일단 "RAM을 사용 안 하다 RAM 읽기"는 "Case2"에 대응된다고 납득하고 넘어가자.
