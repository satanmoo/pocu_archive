## A) RAM 설치하기

![[Pasted image 20251231014651.png]]

우선 최상위 비트(A15)가 1이고, ROM을 사용하는 상황이다.

CPU의 주소핀이 ROM의 주소핀과 매핑됬다.
- CPU에서 "0x8000 ~ 0xFFFF" 주소
- ROM에서 "0x0000 ~ 0x7FFF" 주소
	- [[014. ROM 주소 매핑]], [[015. ROM 설치하기]] 참고
- NAND 게이트로 만든 NOT 게이트로 "A15"와 "CEB"를 연결했다.

그리고 ROM을 선택했기 때문에 RAM의 "CEB"에는 1이 들어와서 RAM이 비활성화 상태여야 한다.

![[Pasted image 20251231020259.png]]

이제 최상위 비트(A15)가 0이고, RAM을 선택했다.

![[Pasted image 20251231020405.png]]

[[020. RAM 설치하기]] 에서 언급했듯이, RAM의 "CEB"는 NOT게이트 없이 그냥 연결해도 괜찮아 보인다.

CPU의 주소와 RAM의 주소는 동일하게 매핑이 된다.
- 최상위 비트가 0이니까!

![[Pasted image 20251231020526.png]]

A15핀을 NAND 게이트로 만든 NOT 게이트를 거치지 않고, 바로 RAM의 CEB와 연결했다.
