## tokenize_malloc()

### 구현

- 동적할당에 필요한 정보
  - 토큰의 개수
  - 각 토큰의 길이

- 문자열 순회
  - 토큰을 담는 배열 동적할당(tokens)
    - 길이 1
    - 초기화 값으로 tokens[0] == NULL
  - remove leading delims
  - 토큰 생성
    - token의 시작 주소 저장
    - dlims 만나면 토큰의 끝 저장
    - 토큰의 길이 구함
    - 토큰 동적 할당
    - 토큰에 문자열 복사
  - tokens에 token 추가
    - tokens 길이 증가
    - malloc, memcpy, free 콤보
    - tokens의 마지막 요소에 NULL을 추가
- 내부에서 동적 메모리 해제할 필요없음
