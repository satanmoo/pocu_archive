## 입력된 단어가 영어사전에 있으면 true 아니면 false를 반환하는 프로그램에서 해시테이블(셋)에 문자열을 저장하면 시간 복잡도는 어떻게 될까?

- 버킷의 슬롯을 찾을 때 O(1)
- 슬롯에서 기본적으로 문자열을 참조가 아닌 content-based로 비교하기 때문에 O(K)
    - K는 입력한 단어의 길이
    - 해시 충돌이 발생해도 슬롯에서 찾음

## 트라이와 다른 자료구조의 비교

- 입력된 단어를 포함하는 단어를 보여주는 프로그램(검색 자동완성)에서 트라이가 유용함

### 이진 트리와 트라이의 비교

- 이진 트리:
    - 노드가 키를 저장
    - 키값(대소 관계)에 따라 각 노드의 연결이 결정
    - 노드 자식 최대 2개

- 트라이:
    - 노드는 한 문자만 저장
        - 노드가 키를 저장하지 않음
    - 노드의 위치가 키를 결정함
    - 노드 자식 수의 제한이 없음

### 해시 테이블과 트라이의 비교

- 자동 완성을 위해서는 트라이가 유리함
    - 공통 접두어

- 해시 테이블에서 문자열을 참조로 저장하기 때문에 트라이의 메모리 사용량이 더 클 수 있음

- 트라이는 해시 테이블보다 최악의 경우가 더 빠름
    - 애초에 최악이라는게 없음

- 문자열 딱 하나를 찾을 때는 해시 테이블에서 해시 충돌이 발생하면 트라이보다 불리함
    - 트라이는 O(k)
    - 해시 테이블은 O(m * k)
        - 여기서 k는 찾을 문자열의 길이
        - 여기서 m은 찾을 문자열이 저장된 버킷의 크기

## 자동 완성 트라이 코드 작성하기

```java
import java.util.ArrayList;
import java.util.Collections;
import java.util.HashMap;
import java.util.List;

public class Program {

    public static void main(String[] args) {
        var trie = new Trie();

        trie.insert("ab");
        trie.insert("abc");
        trie.insert("abcd");

        var result = trie.findOrNull("c");
        result = trie.findOrNull("a");
    }

    final static class Trie {
        private final TrieNode root = new TrieNode();

        void insert(final String str) {
            insertRecursive(str, root, 0);
        }

        void insertRecursive(final String str, final TrieNode node, int depth) {
            if (depth == str.length()) {
                node.isEndOfWord = true;
                return;
            }

            final char c = str.charAt(depth);
            final TrieNode child = node.children.computeIfAbsent(c, k -> new TrieNode());
            insertRecursive(str, child, depth + 1);
        }

        void insertIter(final String str) {
            TrieNode current = root;
            for (int i = 0; i < str.length(); ++i) {
                final char c = str.charAt(i);
                current = current.children.computeIfAbsent(c, (k) -> new TrieNode());
            }
            current.isEndOfWord = false;
        }

        List<String> findOrNull(final String str) {
            final List<String> result = new ArrayList<>();
            final TrieNode found = findNodeOrNullRecursive(root, str, 0);
            if (found == null) {
                return Collections.emptyList();
            }
            findSuffixRecursive(found, new StringBuilder(str), result);
            return result;
        }

        TrieNode findNodeOrNullRecursive(final TrieNode node, final String str, final int depth) {
            if (depth == str.length()) {
                return node;
            }

            final char nextChar = str.charAt(depth);
            final TrieNode nextNode = node.children.get(nextChar);
            if (nextNode == null) {
                return null;
            }

            return findNodeOrNullRecursive(nextNode, str, depth + 1);
        }

        void findSuffixRecursive(final TrieNode node, final StringBuilder str, final List<String> outResult) {
            if (node.isEndOfWord) {
                outResult.add(str.toString());
            }

            for (var next : node.children.entrySet()) {
                str.append(next.getKey());
                findSuffixRecursive(next.getValue(), str, outResult);
                str.deleteCharAt(str.length() - 1);
            }
        }
    }

    final static class TrieNode {
        final HashMap<Character, TrieNode> children = new HashMap<>();
        boolean isEndOfWord = false;
    }
}
```

```java
package academy.pocu.comp3500.lab7.app;

import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

public class Program {

    public static void main(String[] args) {
        final Point p1 = new Point(1, 4);
        final GameObject gameObject1 = new GameObject(p1, 1);

        final Point p2 = new Point(7, 9);
        final GameObject gameObject2 = new GameObject(p2, 2);

        final Point p3 = new Point(5, 5);
        final GameObject gameObject3 = new GameObject(p3, 3);

        final Point p4 = new Point(3, 4);
        final GameObject gameObject4 = new GameObject(p4, 4);

        final Point p5 = new Point(2, 7);
        final GameObject gameObject5 = new GameObject(p5, 5);

        final Point p6 = new Point(9, 3);
        final GameObject gameObject6 = new GameObject(p6, 6);

        Point topLeft = new Point(0, 0);

        BoundingRect rect = new BoundingRect(topLeft, 10, 10);
        final Quadrant root = new Quadrant(rect);

        root.insert(gameObject1);
        root.insert(gameObject2);
        root.insert(gameObject3);
        root.insert(gameObject4);
        root.insert(gameObject5);
        root.insert(gameObject6);

        topLeft = new Point(0, 1);
        rect = new BoundingRect(topLeft, 4, 3);

        List<GameObject> gameObjects = root.getGameObjects(rect);

        print(gameObjects);

        topLeft = new Point(5, 8);
        rect = new BoundingRect(topLeft, 1, 1);

        gameObjects = root.getGameObjects(rect);

        print(gameObjects);

        topLeft = new Point(6, 3);
        rect = new BoundingRect(topLeft, 3, 1);

        gameObjects = root.getGameObjects(rect);

        print(gameObjects);
    }

    private static void print(List<GameObject> gameObjects) {
        System.out.println("--------------------");
        for (int i = 0; i < gameObjects.size(); ++i) {
            GameObject obj = gameObjects.get(i);
            System.out.println(String.format("%d. [%d] (%d, %d)",
                    i + 1,
                    obj.data,
                    obj.position.x,
                    obj.position.y));
        }

        System.out.println(String.format("Count: %d", gameObjects.size()));

        System.out.println("--------------------");
    }

    final static class Quadrant {
        private static final int MIN_QUAD_DIMENSION = 1;

        final BoundingRect boundingRect;
        final ArrayList<GameObject> gameObjects = new ArrayList<>();

        Quadrant leftTopOrNull;
        Quadrant rightTopOrNull;
        Quadrant leftBottomOrNull;
        Quadrant rightBottomOrNull;

        public Quadrant(final BoundingRect boundingRect) {
            this.boundingRect = boundingRect;
            createChild();
        }

        private void createChild() {
            final int width = this.boundingRect.getWidth();
            final int height = this.boundingRect.getHeight();
            final int leftTopX = this.boundingRect.leftTop.x;
            final int leftTopY = this.boundingRect.leftTop.y;

            if (width < 2 * MIN_QUAD_DIMENSION || height < 2 * MIN_QUAD_DIMENSION) {
                // 더 이상 분할할 수 없음
                // 리프 노드
                return;
            }

            {
                var br = new BoundingRect(new Point(leftTopX, leftTopY), width / 2, height / 2);
                this.leftTopOrNull = new Quadrant(br);
            }

            {
                var br = new BoundingRect(new Point(leftTopX + width / 2, leftTopY), width / 2, height / 2);
                this.rightTopOrNull = new Quadrant(br);
            }

            {
                var br = new BoundingRect(new Point(leftTopX, leftTopY + height / 2), width / 2, height / 2);
                this.leftBottomOrNull = new Quadrant(br);
            }

            {
                var br = new BoundingRect(new Point(leftTopX + width / 2, leftTopY + height / 2), width / 2, height / 2);
                this.rightBottomOrNull = new Quadrant(br);
            }
        }

        boolean insert(final GameObject gameObject) {
            final Point position = gameObject.position;

            if (!this.boundingRect.contains(position)) {
                return false;
            }

            this.gameObjects.add(gameObject);
            // 리프 노드 체크
            if (this.leftTopOrNull != null) {
                this.leftTopOrNull.insert(gameObject);
                this.rightTopOrNull.insert(gameObject);
                this.leftBottomOrNull.insert(gameObject);
                this.rightBottomOrNull.insert(gameObject);
            }

            return true;
        }

        // 재귀적으로 BoundingRect가 포함된 가장 크기가 작은 노드의 gameObjects 반환
        List<GameObject> getGameObjects(final BoundingRect rect) {
            if (!this.boundingRect.contains(rect)) {
                return Collections.emptyList();
            }
            // 리프노드
            if (this.leftTopOrNull == null) {
                return List.copyOf(this.gameObjects);
            }

            if (this.leftTopOrNull.boundingRect.contains(rect)) {
                return this.leftTopOrNull.getGameObjects(rect);
            }

            if (this.rightTopOrNull.boundingRect.contains(rect)) {
                return this.rightTopOrNull.getGameObjects(rect);
            }

            if (this.leftBottomOrNull.boundingRect.contains(rect)) {
                return this.leftBottomOrNull.getGameObjects(rect);
            }

            if (this.rightBottomOrNull.boundingRect.contains(rect)) {
                return this.rightBottomOrNull.getGameObjects(rect);
            }

            return List.copyOf(this.gameObjects);
        }
    }

    final static class Point {
        final int x;
        final int y;

        Point(int x, int y) {
            this.x = x;
            this.y = y;
        }
    }

    final static class BoundingRect {
        final Point leftTop;
        final Point rightBottom;

        public BoundingRect(final Point leftTop, final int width, final int height) {
            this.leftTop = leftTop;
            this.rightBottom = new Point(leftTop.x + width, leftTop.y + height);
        }

        int getWidth() {
            return rightBottom.x - leftTop.x;
        }

        int getHeight() {
            return rightBottom.y - leftTop.y;
        }

        boolean contains(final Point point) {
            return (point.x >= leftTop.x && point.x <= rightBottom.x) && (point.y >= leftTop.y && point.y <= rightBottom.y);
        }

        boolean contains(final BoundingRect other) {
            final Point otherLeftTop = other.leftTop;
            final int otherRightBottomX = otherLeftTop.x + other.getWidth();
            final int otherRightBottomY = otherLeftTop.y + other.getHeight();
            return otherLeftTop.x >= this.leftTop.x &&
                    otherLeftTop.y >= this.leftTop.y &&
                    otherRightBottomX <= this.rightBottom.x &&
                    otherRightBottomY <= this.rightBottom.y;
        }
    }

    final static class GameObject {
        final Point position;
        final int data;

        public GameObject(final Point position, final int data) {
            this.position = position;
            this.data = data;
        }
    }
}
```
