## 더 많은 자릿수를 계산하는 경우

![[Pasted image 20260127010859.png]]

![[Pasted image 20260127010920.png]]

![[Pasted image 20260127011030.png]]

이 마법의 회로가 "COMP1000"에서 배운 *Half Adder*

*S2* 의 경우 2bit 의 값이다.

빨간색으로 표시된 물음표는 무엇일까 캐리비트 같은데
- *C1* 이 1이고, *A2 ^ B2*가 1인 경우
	- *A2*가 1, *B2*가 0
	- *A2*가 0, *B2*가 1
		- 캐리가 발생하긴 하네요. 근데 *C1*, *A2*, *B2* 모두 1이면???

![[Pasted image 20260127012259.png]]

이 두 *AND 게이트* 는 캐리를 계산한다.

아래 *AND* 게이트는 *A2*, *B2* 모두 1인 경우 출력이 1이다.

![[Pasted image 20260127012435.png]]

*C1*, *A2*, *B2* 모두 1인 경우 받아 올림을 나타내기 위해 이 두 *AND 게이트* 의 출력을 *OR 게이트* 로 보내자

![[Pasted image 20260127012531.png]]

*OR 게이트* 의 출력(*C1*, *A2*, *B2* 캐리)을 위와 비슷하게 *XOR 게이트* 로 보내면 된다.
