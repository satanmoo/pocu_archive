![[Pasted image 20260111032047.png]]

내부 연산은 실제 하드웨어(6502)의 동작을 반영했다

"Excute ADC"
- A: A 레지스터
- M: 메모리
- A 레지스터의 값과 메모리의 값(`0x06`)을 더한다.
- 여기서 중요한 것은 3사이클에서 A 레지스터의 값이 바로 업데이트 되지 않는다. ALU에 남아있다.

"ALU -> A"
- ALU에 남아있던 값을 A레지스터에 썼다.
- 4사이클

참고로 에뮬레이터에서는 3사이클에 바로 A레지스터가 업데이트 된다.
