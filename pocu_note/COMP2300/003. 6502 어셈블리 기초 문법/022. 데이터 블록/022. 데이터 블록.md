## 데이터 블록

![[Pasted image 20260118215548.png]]
`.BYTE`: 1바이트 크기, 바이트 단위로 데이터를 저장한다.
```asm
.BYTE $00,$01,$02,$03
```

메모리에 어떤 주소부터 `0x00`, `0x01` ... 연속으로 1바이트 크기로 값을 저장한다.

```asm
.BYTE "Hello World"
```

각 문자는 1바이트로 메모리에 연속적으로 저장된다.

```asm
.BYTE 0 ; 널 문자
```

C처럼 널 문자를 넣어 문자열의 마지막임을 표시했다.

`.WORD`: 2바이트 크기 단위로 데이터를 저장한다.
- 리틀 엔디언이 적용된다.

```asm
.WORD $1000,$1001
```

리틀 엔디언이니까 `0x00 0x10` 순서로 저장된다. 이하 동일

![[Pasted image 20260118220227.png]]

어셈블된 결과를 확인해보자

`.BYTE` 지시문에 따라 데이터를 1바이트 단위로 저장하기 때문에 엔디언 문제는 발생하지 않는다.

초록색으로 하이라이트한 부분은 `.WORD`  지시문에 따라 2바이트 단위로 데이터를 저장해 리틀 엔디언을 확인할 수 있다.

`18 AD ....` 여기서 부터는 이하 명령어다.

### 코드 해석

![[Pasted image 20260118221241.png]]

우선 [[003. 헬로 16 코드 작성 및 어셈블#헬로 16! (1)]]에서 봤듯이 생략된 지시문을 기억해야 한다.
- `.ORG 8000`

```asm
.BYTE $00,$01,$02,$03
```

주소 `0x8000` : 값 `0x00`
주소 `0x8001` : 값 `0x01`
주소 `0x8002` : 값 `0x02`
주소 `0x8003` : 값 `0x03`

이렇게 저장한다.

```asm
clc
lda $8000
adc $8001
adc $8002
adc $8003
sta $1000
```

*A 레지스터*에 주소 `$8000`의 값(`0x00`)을 불러온다.
*A 레지스터*의 값(`0x00`)에 주소 `$8001`의 값(`0x01`)을 더하고 Carry 비트 값(`0x00`)을 더해서 *A 레지스터*에 저장한다.
...

이 결과 *A 레지스터*에는 1부터 3까지 합이 저장된다.

주소 `$1000`에 *A 레지스터*의 값(`0x06`)을 저장한다.

- [[004. 헬로 16 바이너리 파일 생성#리셋 벡터 적기]]

![[Pasted image 20260118222609.png]]

리섹 벡터(`$FFFC`, `$FFFD`)의 값이 `0x8000`다. 따라서 CPU는 ROM의 `$8000`주소의 값을 읽어서 프로그램을 시작한다.

`$8000`의 주소에 있는 값은 `0x00`이다. 이는 `BRK`라는 opcode다.

의도한 대로 1부터 3까지 더 하도록 프로그래밍을 하지 못했다. 어떻게 고치면 될까?

1. 초록색으로 하이라이트한 `.WORD $8000`을 수정한다. `$8000`부터가 아니라 초록색으로 하이라이트한 `clc`가 시작하는 `$8012`로 바꾸면 된다.
2. `clc`위에 `.ORG` 지시문을 추가한다. 예를 들어 `.ORG $C000, 0`이렇게 추가하고, 아래의 `.WORD $C000`로 수정하면 된다.
	- 본질적으로 `clc`의 주소를 리셋벡터로 만드는 것!
3. `clc` ~ `sta`까지 코드를 위로 올리고 `.BYTE` ~ `.WORD`를 아래로 내리고, 아래로 내린 것에 `.ORG`를 박고 명령어의 피연산자 주소값을 수정하기

사실 이 방법도 `$C000`이라는 매직 넘버를 사용하기 때문에 좋지 않다. 따라서 *라벨*을 사용하자.

## 라벨은 데이터 블록의 친구!

![[Pasted image 20260118224730.png]]

라벨은 위치한 곳의 바로 **아래**의 바이트 주소를 가리키는 별칭이다.

`nums`는 `.ORG $C000,0` + `.BYTE $00` 때문에 `$C0000`의 주소에 해당하는 별칭이다.

`hello`는 `$C000` + 4 의 주소에 해당하는 별칭이다.

`sum`은 [[021. 상수, 포인터 변수#상수 - 상수형 변수]]를 사용했다.
